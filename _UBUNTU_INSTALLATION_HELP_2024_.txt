For me on ubuntu:
Something like this:

For beginning (also for when your packages are messed up which seems to happen every time):
cd /var/www/YOUR_WEBSITE_NAME_HERE
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
nvm install --latest-npm
npm i -g pm2

For updating:
cd /var/www/YOUR_WEBSITE_NAME_HERE
npm run build
pm2 kill
pm2 start --name=YOUR_WEBSITE_NAME_HERE npm -- start

Helpful when your pm2 inevitably crashes:
pm2 log --lines 1000
